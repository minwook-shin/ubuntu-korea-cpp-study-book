# 헤더 파일

## 해더파일이란

함수들을 선언해둔 곳입니다.

기능을 구현하는 곳은 cpp로 보내야합니다.

## 이렇게까지 하는 이유

중복 정의를 피해서 관리, 공유가 쉽다.

## 만들어보기

* sum.h

```c++
int sum(int a,int b);
```

* sum.cpp

```c++
#include "sum.h"
int sum(int a, int b){
    return a + b;
}
```

* main.cpp

```c++
#include "sum.h"
#include <iostream>

using namespace std;

int main ()
{
    int result = int sum(5,10);
    cout<<"result :  "<< result <<endl;
    return 0;
}
```

## 클래스

클래스 형식도 public, private을 헤더파일에 선언해주면 됩니다.
